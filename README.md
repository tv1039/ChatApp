# SwiftUI 채팅앱 구현 및 테스팅...

## Overview
Firebase를 활용한 채팅앱을 SwiftUI로 구현하고 테스팅하는 것을 목표로 합니다. </br>
주요 기능은 사용자 인증, </br>
실시간 데이터베이스, </br>
파일 저장소입니다. </br>

## Technologies Used
- **Programming Languag**: </br> Swift
- **Framework**: </br> SwiftUI
- **Backend Services**: </br> Firebase
  - **Auth**: 사용자 인증
  - **Firestore**: 실시간 데이터베이스
  - **Storage**: 파일 저장소
